One-time setup. After this, just update your Google Sheet and click Refresh.

1
Set up the Google Sheet
Go to sheets.google.com → Import → upload Solutions_Knowledge_Base.xlsx
Make sure the tab at the bottom is named Solutions Database
2
Set up Google Apps Script
Go to script.google.com → New project → delete default code → paste apps_script_backend.js
Replace YOUR_ANTHROPIC_API_KEY_HERE with your key from console.anthropic.com
Replace YOUR_GOOGLE_SHEET_ID_HERE with your Sheet ID
Click Deploy → New deployment → Web app → Execute as: Me → Anyone → Deploy
Copy the Web App URL
3
Paste your Apps Script URL below and connect
